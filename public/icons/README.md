# Favicon & Icon Set

Placeholder folder. Before deploy, generate the full favicon set from your logo:

**Required files:**
- `favicon.ico` (multi-size, 16 + 32 + 48)
- `icon-32.png` (32×32)
- `icon-192.png` (192×192)
- `icon-512.png` (512×512)
- `icon-maskable.png` (512×512, safe zone in center)
- `apple-touch-icon.png` (180×180)

**Easy way:** Upload your logo to https://realfavicongenerator.net/ — it produces the whole set. Drop the files here.

**Design suggestion for the logo:**
- Yellow (#FFE135) background
- Bold black "AD" or the italic "Amazing" wordmark
- Rounded square shape for maskable variant

Until you replace these, browsers will show a default icon (harmless, but ugly).
